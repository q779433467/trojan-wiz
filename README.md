---
# 安装命令：
> wget -N --no-check-certificate https://github.com/q779433467/trojan-wiz/raw/master/ins.sh && chmod +x ins.sh && sudo bash ins.sh
---

- ubuntu 16.04+
- debian 9(理论上应该支持debian 8)
- centos 7+
